# Common Platform

## Shared technical assumptions

- Vehicle platform compatibility with Bajaj RE form factor.
- Shared electrical architecture conventions: 48–72V bus, dual-battery BMS.
- Safety and certification assumptions to be tracked centrally.

## Shared services

- Data collection and simple telemetry (SOC, temp, transfer events).
- Configuration management and firmware updates over workshop tools.
- Supplier and BOM coordination.

## Platform decisions to confirm

- Battery chemistry (LFP recommended) and thermal strategy.
- Communication and monitoring approach (minimal telemetry in pilot).
- Installation standards and mounting specifications.
