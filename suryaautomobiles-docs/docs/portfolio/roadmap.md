# Roadmap

## Phase 0 — Validation

- Confirm concept assumptions.
- Capture initial target users and operating regions.
- Gather early evidence from pilots and conversations.

## Phase 1 — Prototype

- Build proof-of-concept hardware.
- Validate solar charging, battery transfer, and safety assumptions.
- Document technical gaps and supplier dependencies.

## Phase 2 — Pilot

- Run limited vehicles in the chosen geography.
- Collect field data and driver feedback.
- Refine pricing, installation workflow, and support model.

## Phase 3 — Scale

- Prepare for repeatable installation.
- Expand to additional fleets and markets.
- Formalize manufacturing and certification strategy.
