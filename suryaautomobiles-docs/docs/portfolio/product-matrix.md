# Product Matrix

## Product family overview

| Product | Target user | Core value | Key assumptions |
|---|---|---|---|
| Charging stations | Fleet operators and public charging partners | Fast deployment and usage visibility | Shared power and safety standards |
| Retrofit kits | Retrofit workshops | Low-friction installation path | Compatible with common vehicle platforms |
| Range extenders | Driver-facing retrofit users | Daily range extension and reduced charging stops | Solar-assisted reserve energy |

## Notes from product story

- Recommended panel for range-extender: Waaree 150W semi-flexible (good fit & daily yield ~0.70 kWh -> ~48 km/day).
- Reserve battery: 1 kWh LFP recommended to minimize wasted energy and maximize practical range.
- BOM and cost targets: aim for an ASP around ₹25,000 with a clear supplier shortlist and installation workflow.
