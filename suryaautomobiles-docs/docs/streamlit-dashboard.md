# Streamlit Pricing Dashboard

A simple Streamlit dashboard is available in the repository to estimate a price point for the SuryaChakra product bundle.

## What it calculates

The dashboard uses a few simple assumptions to estimate the commercial price that would make sense for the system:

- Retrofit hardware cost
- Portable charger cost
- Solar range extender cost
- Target markup
- Service margin
- Estimated monthly customer savings
- Target payback period

## How to run it locally

From the repository root, run:

```bash
streamlit run price_dashboard.py
```

## Why it matters

This dashboard helps the team test pricing scenarios quickly before deciding on an offer for customers or fleet partners.
