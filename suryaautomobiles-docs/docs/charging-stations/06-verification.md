# 06. Verification

## Test strategy

- Lab functional testing: MPPT charging curves, transfer relay pre-charge, BMS SOC accuracy.
- Safety testing: thermal, short-circuit, and isolation tests.
- Field pilot: deploy to 20–50 vehicles and measure daily yield, reliability, and driver feedback.

## Acceptance criteria

- Average daily delivered solar energy >= 0.4–0.7 kWh (target ~0.7 kWh for Waaree 150W panel).
- Typical daily range added >= 40 km in pilot conditions.
- Transfer reliability >99% across pilot fleet.
- No critical thermal or safety incidents in field trials.

## Test artifacts

- Test logs, telemetry exports, and photographic installation evidence for each pilot vehicle.
