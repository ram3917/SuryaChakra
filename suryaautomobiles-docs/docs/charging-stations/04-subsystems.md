# 04. Subsystems

## Subsystem breakdown

- Power handling: MPPT, pre-charge, transfer relay, fuses.
- Monitoring and control: BMS, SOC monitoring, telemetry.
- User and operator interface: installation indicators, operator checklists.
- Installation and maintenance workflow: mechanical mounts, wiring harness, and test points.

## Responsibilities

- Hardware team: design panels, enclosures, and power electronics.
- Field team: installation, diagnostics, and feedback.
- Supplier management: sourcing cells, panels, and MPPT units.
