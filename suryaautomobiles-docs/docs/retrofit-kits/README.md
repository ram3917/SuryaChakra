# Retrofit Kits

## Purpose

Retrofit kits convert common vehicles (Bajaj RE) to support SuryaExtend modules with a minimal installation footprint.

## Target vehicles

- Bajaj RE and other common 3-wheeler platforms used in Telangana and Andhra Pradesh.

## Installation approach

- Lightweight frame mounts for solar panel
- Wiring harness with labelled connectors
- Pre-configured BMS and MPPT modules

## Required components

- Solar panel (Waaree 150W recommended)
- MPPT charger
- 1 kWh LFP reserve battery pack
- Mounting hardware and wiring

## Safety and compliance notes

- Ensure isolation from main drive circuits during installation.
- Follow regional retrofit guidelines and certification steps.
