# Retrofit Kits

## Purpose

Retrofit kits transform a standard vehicle into a solar-assisted, modular electric platform. This guide walks through the system as it is built, from the first fitment decision to the final field acceptance check.

## Step 1: Choose the base vehicle

- Start with a durable three-wheeler platform such as a Bajaj RE.
- Confirm the vehicle has enough roof area, mounting points, and service access for the new hardware.
- Check that the operator can support routine inspection and basic maintenance.

## Step 2: Define the retrofit architecture

- Add a solar panel, MPPT charger, reserve battery, and wiring harness.
- Keep the installation compact so the vehicle remains practical for daily use.
- Separate the high-voltage energy path from the low-voltage control path.

## Step 3: Mount the physical hardware

- Fit the panel mounts to the roof structure using reinforced brackets.
- Route the wiring in protected channels and label every connector.
- Secure the battery enclosure in a location that is stable, ventilated, and easy to inspect.

## Step 4: Install the power electronics

- Connect the solar panel to the MPPT controller.
- Connect the reserve battery through a BMS that provides cell balancing and protection.
- Add a simple transfer or control interface so the operator can switch between normal and reserve power modes.

## Step 5: Validate safety and operation

- Verify isolation from the main drive circuits.
- Confirm that the system can charge, hold, and discharge safely under expected load.
- Record the final wiring map, fuse values, and maintenance notes for the fleet.

## User interview insights

The following notes reflect a structured interview with a pilot operator and a vehicle owner.

- Question: What is the biggest pain point in the current retrofit process?
  - Response: The installation is often slow because the wiring layout and mounting points are not standardized.
- Question: What matters most after installation?
  - Response: Reliability, easy inspection, and safe operation during daily use.
- Question: What would make the retrofit more attractive to fleet users?
  - Response: A simpler pack, clearer labels, and a maintenance checklist that can be followed by local technicians.
- Question: Which feature should be included in the first production version?
  - Response: A compact solar-assisted reserve system with basic safety monitoring and easy service access.

## Final system outcome

The retrofit kit is not just a collection of parts. It is a practical vehicle system that combines mounting, power conversion, battery storage, and operator guidance into one repeatable package.
